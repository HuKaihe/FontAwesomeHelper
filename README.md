## 开始您的FontAwesome Helper之旅

FontAwesome Helper简约的设计风格在界面和功能上都得到了完美的体现。

![工作区.png](https://upload-images.jianshu.io/upload_images/5494319-2253442ebde95512.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

系统为图标管理提供了简约有效的功能：

1. 图标管理
2. 图标集合管理
3. 用户管理

不如让我们花3分钟的时间快速体验一下FontAwesome Helper的主要功能吧：

##### 新增加一个集合

点击左侧图标集合控制区最下面的【+】号图标，输入集合名称，按下回车或鼠标点击工作区的任意位置即可完成新增集合的操作。

![新增集合.png](https://upload-images.jianshu.io/upload_images/5494319-26b9ba8e8dd18a6a.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

我们新增加一个叫做hello world的集合。

##### 将图标放置到集合内

之后，当鼠标悬浮到图标管理区中任意图标下面的购物车上，就会出现一个自定义集合的列表，单机集合名称就可以把这个图标添加到该集合中了。如果你还没来得及创建一个新的集合，可以点击购物车的图标，将这个图标加入到【我的收藏】中。

![添加图标.png](https://upload-images.jianshu.io/upload_images/5494319-b4caaf4138d50149.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

##### 同步数据

在没有登录的情况下，FAH会将数据存储到浏览器的localstorage中，为保证你可以在任何地方获得想要的数据集合，我们强烈推荐你登录自己的账号。

你可以把本地的数据存储想象成一个只存在于你现用浏览器上的本地账号，可以在登陆后手动对两个账号的数据进行同步操作：

![数据同步.png](https://upload-images.jianshu.io/upload_images/5494319-7bb61144f5ba77bf.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

但这些操作非常危险，它会使你账号存储在云端的数据完全被本地覆盖（或者反言之），且不可撤销。

## 功能列表

### 一、图标集合管理

| 功能名称 | 操作 | 备注 |
| :--- | :--- | :--- |
| 新增集合 | 点击左侧图标集合控制区最下面的【+】号图标, 输入集合名称<br/>按下回车或鼠标点击工作区的任意位置即可完成新增集合的操作。| 1. 集合名称不得超过15个字符 <br/> 2. 集合名称可重复 <br/> |
| 删除集合 | 鼠标悬浮到集合上，点击左侧的【删除】图标 | 1. 一旦删除不可恢复请谨慎操作 <br/>2.【全部图标】和【我的收藏】不可操作|
| 集合重命名 | 双击集合名称即可进行修改 | 1. 命名规则与【新增集合一致】<br/> 2.【全部图标】和【我的收藏】不可操作 |
| 调整集合顺序 | 对图标集合进行拖动 | 无 |
| * 批量添加图标至集合 | 单击【全部图标】中图标分类名称后的购物车标志 | 无 |

### 二、图标管理

| 功能名称 | 操作 | 备注 |
| :--- | :--- | :--- |
| 新增图标至集合 | 当鼠标悬浮到图标管理区中任意图标下面的购物车上<br/>就会出现一个自定义集合的列表，单机集合名称就可以把这个图标添加到该集合中了。<br/>如果你还没来得及创建一个新的集合，可以点击购物车的图标<br/>将这个图标加入到【我的收藏】中。| 自定义集合页面也可以将图标添加到别的集合 |
| 删除图标 | 鼠标悬浮到自定义集合的图标上<br/>点击左上角的【删除】标志 | 1. 一旦删除不可恢复请谨慎操作 <br/>2.【全部图标】和【查找列表】不可操作|
| 快速复制图标名称 | 单击任意图标 | 图标名称直接复制到剪贴板上，在别处直接粘贴即可 |
| * 批量添加图标至集合 | 单击【全部图标】中图标分类名称后的购物车标志 | 无 |

### 三、用户管理

| 功能名称 | 操作 | 备注 |
| :--- | :--- | :--- |
| 用户登录 | 在页面右上角输入用户名和密码<br/>单击【登录】按钮| 无 |
| 用户注册 | 单击页面右上角【注册】按钮<br/>完善相关信息 | 1.用户名一旦确定不可修改 <br/>2.全站用户名不可重复<br/>3.用户名最长20个字符<br/>4.请填写正确的邮箱，我们会通过这个邮箱联系你<br/>5.密码复杂程度没有任何限制，一旦设定，不可更改<br/>|
| 用户注销 | 用户登陆后单击用户名旁边的【注销】图标 | 无 |
| 同步本地数据至云端 | 用户登陆后单击用户名旁边的【上传】图标 |这是个非常危险的操作<br/>它会使你账号存储在云端的数据完全被本地覆盖），且不可撤销 |
| 同步云端数据至本地 | 用户登陆后单击用户名旁边的【同步】图标 |这是个非常危险的操作<br/>它会使你本地的数据完全被账号的数据覆盖，且不可撤销 |
| 分享数据 | 用户以本地存储作为中间层，对两个账号进行【同步】和【上传】操作 |这是个操作的技巧，系统本身不提供数据分享的功能|


### 三、其它

| 功能名称 | 操作 | 备注 |
| :--- | :--- | :--- |
| 邮件 | 用户注册完毕后会收到内容为【致用户的一封信】的电子邮件| 无 |
| 数据收集 | 系统会收集用户的一些行为数据以帮助我们更好的对系统进行完善，包括但不限于：<br/> 1. 用户注册时间 2. 用户上次登录时间<br/> 3. 用户登陆次数<br/> | 本站架构上对安全方面没有更多处理，所以请您在使用（尤其在注册时）时<br/>不要在账号里暴露自己的真实姓名， 证件号码与常用密码 |
